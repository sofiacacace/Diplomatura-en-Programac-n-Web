Tabla de descripción de los diferentes tipos de inputs en formularios:


  1. Ir a tabla con descripción de tipos de entradas (formularios).

  2. Ir a ejemplos de formularios.


Tipo Descripción Ejemplo
Editor Vista
button Define un botón en el que se puede hacer clic (se usa principalmente con JavaScript para activar un script). 
<input type="button" onclick="alert('Hola mundo!')" value="Has click!">
checkbox Define una casilla de verificación con forma de cuadrado. 
<form>
                    <label>Comida favorita:</label><br>      
                    <input type="checkbox" id="comida1" name="comida1" value="hamburguesas">
                    <label for="comida1"> Hamburguesas</label><br>
                    <input type="checkbox" id="comida2" name="comida2" value="pastas">
                    <label for="comida2"> Pastas</label><br>
                    <input type="checkbox" id="comida3" name="comida3" value="sushi">
                    <label for="comida3"> Sushi</label>
                  </form>



color Define un campo para seleccionar un color (sistema RGB). 
<input type="color" id="favcolor" name="favcolor" value="#ff00ff">
date Define un control de fecha (año, mes, día (sin hora)). 
<label for="fnacimiento">Nacimiento:$lt/label>
<input type="date" id="fnacimiento" name="fnacimiento">
datetime-local Define un control de fecha y hora (año, mes, día, hora (sin zona horaria)). 
<label for="fyhnacimiento">Nacimiento (fecha y hora):</label>
<input type="datetime-local" id="fyhnacimiento" name="fyhnacimiento">
email Define un campo para una dirección de correo electrónico. 
<label for="email">Enter your email:</label>
<input type="email" id="email" name="email">
file Define un campo de selección de archivos y un botón "Examinar" (para cargas de archivos). 
<label for="file">Seleccionar un archivo:</label>
<input type="file" id="file" name="file">
hidden Define un campo de entrada oculto. 
<input type="hidden" id="password" name="password" value="123ABC">
image Define una imagen como botón de envío. 
<input type="image" name="image-button" src="URL" alt="click para enviar">
month Define un control de mes y año (sin zona horaria). 
<label for="month&year">Ingrese mes y año de vencimiento:</label>
<input type="month" id="expire" name="expire">
number Define un campo para ingresar un número. 
<label for="quantity">Ingrese su edad:</label>
<input type="number" id="quantity" name="quantity" min="18" max="80">
password Define un campo para ingresar una contraseña. 
<label for="pwd">Password:</label><br>
<input type="password" id="pwd" name="pwd"><br><br>


radio Define una casilla de verificación con forma de círculo. 
<form>
      <label>Género:</label><br><form action="/action_page.php">
      <input type="radio" id="male" name="gender" value="masculino">
      <label for="male">Male</label><br>
      <input type="radio" id="female" name="gender" value="femenino">
      <label for="female">Female</label><br>
      <input type="radio" id="other" name="gender" value="otro">
      <label for="other">Other</label><br><br>
</form>




range Define un control de rango. 
<label for="vol">Volumen (entre 0 y 50):</label>
  <input type="range" id="vol" name="vol" min="0" max="50">
reset Define un botón de reseteo. 
<input type="reset">
search Define un campo de texto para ingresar una búsqueda. 
<label for="gsearch">Search Google:</label>
<input type="search" id="gsearch" name="gsearch">
Submit Define un botón de envío. 
<input type="submit" value="submit">
tel Define un campo para ingresar un número de teléfono. 
<label for="phone">Número de teléfono:</label><br>
<input type="tel" id="phone" name="phone" placeholder="123-45-678" pattern="[0-9]{3}-
[0-9]{2}-[0-9]{3}" required><br><br>


time Define un control para ingreso de hora. 
<label for="appt">Seleccionar hora:</label>
 <input type="time" id="appt" name="appt">
text Opción por default. Define una única línea de texto. 
<input type="text">
url Define un campo para ingreso de URL. 
<label for="homepage">Ingresa tu homepage:</label>
<input type="url" id="homepage" name="homepage">
week Define un control de ingreso de semana y año. 
<label for="week">Selecciona semana:</label>
<input type="week" id="week" name="week">
