Etiquetas

Elemento HTML: <etiqueta>contenido</etiqueta>

Atributo: se utiliza junto con las etiquetas para ampliar y/o complementar su funcionamiento.


- Estiquetas estructurales:

Etiqueta Descripción
<!DOCTYPE html> Indica que el documento es del tipo html 5.
<html></html> Contiene todo el contenido de la página web.
<head></head> Contiene los metadatos del documento.
<title></title> Título del documento.
<body></body> Contiene el contenido visible por el usuario.

- Estiquetas de texto:

Etiqueta Descripción
<h1></h1> ... <h6></h6> Encabezados desde el tipo 1 hasta el 6. A mayor número menor tamaño e importancia.
<p></p> Contienen los párrafos.
<blockquote></blockquote> Texto citado desde fuente externa.
<em></em> Texto enfatizado.
<strong></strong> Texto importante.

- Estiquetas de lista:

Etiqueta Descripción
<ul></ul> Lista desordenada con viñetas.
<ol></ol> Lista ordenada con numeración.
<li></li> ítems de una lista.

- Estiquetas de enlace:

Etiqueta Descripción
<a></a> Enlace a un recurso: <a href="URL">texto</a>
<img></img> Se "enlaza" una imagen y se muestra en el documento: <img src="URL">

NOTAS:

1) Las rutas pueden ser relativas (se declaran a partir del lugar donde se encuentra el archivo html), o absolutas (comienzan desde el origen de carpetas)

2) La etiqueta <a></a> se utiliza también para crear enlaces a diferentes secciones de una misma página. A esto se lo llama anclas.


- Estiquetas de tabla:

Etiqueta Descripción
<table></table> Indica comienzo y fin de una tabla.
<tr></tr> Indica comienzo y fin de una fila.
<td></td> Indica comienzo y fin de una columna.
<th></th> Se utiliza para los encabezados de una tabla.

- Estiquetas contenedoras:

Etiqueta Descripción
<div></div> Es una división del documento. Contenedor de otros elementos.
<span></span> Contenedor genérico.

CSS - Hoja de estilos

Selectores

selector { propiedad : valor; }


- Nombre de etiqueta html:

El selector es el nombre de una etiqueta html, por ejemplo h1.


- Id:

El atributo id identifica a una etiqueta en particular. Este identificador debe ser único en todo el documento. Además de dar estilo también se utiliza como identificador de anclas, entre otros elementos. En el documento CSS se escribe anteponiento un numeral (#).


- Class:

Una clase es la definición de un estilo que puede aplicarse a etiquetas html. Se puede añadir una o más clases a una misma etiqueta y a diferencia del id, una misma clase puede estar incluida en más de una etiqueta. En el documento CSS se escribe anteponiento un punto (.).


Estilos

- Estilos principales:

Propiedad Descripción
font-size Se utiliza para indicar tamaño de fuente. Se puede expresar en medidas absolutas o relativas.
font-family Establece la fuente de un texto.
color Establece el color de la fuente de un texto.
background-color Establece el color de fondo de un elemento.
text-align Establece la alineación de un texto.
width Establece el ancho de un elemento. Se puede expresar en medidas absolutas o relativas.
height Establece la altura de un elemento. Se puede expresar en medidas absolutas o relativas.

- Posición de elementos:

Existen muchas formas de posicionar elementos dentro del diseño de una página web. Los más básicos son:

  • Elementos en bloque y en línea:
    • Para elementos en bloque (el elemento se coloca de manera vertical) se usa la propiedad "display: block;"
    • Para elementos en línea (el elemento se coloca de manera horizontal) se usa la propiedad "display: inline;"

  • Flexbox: método para distribuir el espacio entre ítems y mejorar así la alineación.
    1. Colocar al contenedor de elementos la propiedad "display: flex; o display: inline-flex;".
    2. Definir dirección de los elementos con la propiedad "flex-direction: row;" (fila) ó "flex-direction: column;" (columna).
    3. (Opcional) Ajustar los espacios entre elementos:
      • De forma horizontal con "justify-content: flex-start | flex-end | center | space-between | space-around;".
      • De forma vertical con "align-items: flex-start | flex-end | center | space-between | space-around | stretch;".

- Pseudo-clases:

Definen el estilo de un elemento sólo cuando el usuario realiza determinada acción.

Sintaxis: "selector :pseudo-class { propiedad : valor; }"


- Media queries:

Método para incorporar diseño responsivo o sensible a nuestra página web. Es decir, se trata de una técnica que permite hacer webs adaptables a distintos tamaños de pantalla de dispositivos.

Sintaxis: "@media not|only mediatype and (caracteristica-media and|or|not caracteristica-media){ CSS-código }"

Ejemplo:

@media only screen and (orientation: landscape) {
   body {
        background-color: lightblue;
  }
}


Conexión entre CSS y HTML

- En etiquetas html:

Se coloca el estilo en línea usando el atributo: style='propiedad: valor;'.


- En documento html:

En la sección "head" del documento html se añade el bloque <style></style> el cuál incluirá cada selector con su respectivo estilo.

- Hojas de estilos:

Se crea una hoja de estilo (por ejemplo style.css) y se la enlaza en el documento html en la sección "head" utilizando el siguiente link: <link rel="stylesheet" href="style.css">.
